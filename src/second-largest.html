Challenge: Find the Second Largest Number

Summary

Write a function that finds the second largest number in an array of integers

Requirements

You are given an array of integers with at least tow elements

Implement a function to find the second largest number in the array and return it.

Test Cases

Input: [5, 2, 8, 3, 7]
Output: 7

Input: [10, 20, 5, 30, 15]
Output: 20
