Challenge: Validate Parentheses

Summary

Write a function to determine if a given string if parentheses (including curly braces and square brackets) is valid

Requirements

You are given a string containing only parentheses, curly braces, and square brackets

Implement a function to check if the parentheses are balanced and return true if they are, or false if they are not.

Test Cases

Input: "{[()()]}"
Output: true;

Input: "[{()}[{}])]"
Output: false
