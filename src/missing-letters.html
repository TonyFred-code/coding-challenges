Challenge: Missing Letters

Summary

Write a function that finds the missing letter(s) in a given sequence of consecutive letters.

Requirements

You are given a string containing a sequence of consecutive lowercase letters (in alphabetical order), with one (or more) letter(s) missing

Implement a function to find the missing letter(s) and return it.

Test Cases

Input: "abcdefh"
Output: "g"

Input: "jklmno"
Output: "p"
